# BedProx **[WIP]**
Simple Minecraft Bedrock reverse proxy

## Features
- [x] Reverse Proxy
- [x] HAProxy Protocol Support (NOT TESTED)
- [ ] Webhooks
- [ ] REST API

## How to use/deploy
```
git clone https://github.com/haveachin/bedprox
cd bedprox
docker-compose up -build
```
Config can be found in the data/bedprox folder
